ShopWiseMate is a price comparison tool designed to streamline online shopping by aggregating prices from multiple e-commerce platforms. Built with Python and Flask, it provides a user-friendly interface and real-time price updates, helping users make informed purchasing decisions.

## Features

- **Price Comparison**: Aggregates product prices from various online stores for easy comparison.
- **Real-Time Updates**: Ensures users get the latest prices through frequent data scraping.
- **User-Friendly Interface**: Clean, responsive design for seamless user experience.
- **Search Functionality**: Allows users to search for products and view detailed price breakdowns.
- **Data Storage**: Saves search history and user preferences for personalized experiences.

## Technologies

- **Python**: The primary language for backend logic and web scraping.
- **Flask**: Lightweight web framework for creating the application backend.
- **HTML/CSS**: For designing a responsive and interactive frontend.
- **SQLite**: Database for storing product information and user preferences.
- **BeautifulSoup**: Library used for web scraping to gather product data.

## Getting Started

To get started with ShopWiseMate, clone the repository and install the required dependencies:

```bash
git clone https://github.com/kimaninyutu/ShopWiseMate.git
cd ShopWiseMate
pip install -r requirements.txt
```
Once the dependencies are installed, you can run the application:

```bash
python app.py

```
## Deployment

ShopWiseMate can be deployed on cloud platforms like Heroku or AWS. Simply configure the environment and deploy the Flask application.
